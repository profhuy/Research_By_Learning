# GitHub Setup Cho Huy

## 1. Ai nen tao repo

Huy nen la nguoi tao GitHub repo nay.

Ly do:

- Huy la nguoi gom ban cuoi.
- Huy giu file master.
- Huy la nguoi review pilot truoc khi cho sang full.
- Huy can nhin toan bo lich su commit de theo doi tien do.

## 2. Ten repo goi y

Chon mot ten ro rang, vi du:

- `rbl4-bug-report-quality-gate`
- `rbl-team-bug-report-eval`
- `rbl4-gpt4omini-bug-report`

## 3. Trong repo can co gi ngay tu dau

## 4. Cau truc commit nen dung

Moi nguoi commit dung phan minh phu trach.

Vi du:

- `feat: add pilot sample and pilot ground truth`
- `feat: add pilot llm output and api log`
- `feat: add pilot kappa analysis`
- `docs: update notes and result wording`
- `feat: add full experiment output`

Khong nen commit kieu:

- `update`
- `final`
- `done`

## 5. Luat GitHub nhom nen thong nhat

- Khong tu y doi prompt.
- Khong tu y doi rubric.
- Khong tu y doi threshold.
- Khong xoa output loi; neu loi thi danh dau `INVALID`.
- Co batch lon thi commit ngay de tranh mat du lieu.

## 6. Huy can nhan nhom gi ve GitHub

`Minh se tao GitHub repo chung cho RBL-4. Moi nguoi chi sua dung file phan minh phu trach, khong tu doi prompt/rubric/threshold. Lam xong thi commit voi message ro rang roi bao minh de minh review.`
